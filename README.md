## Features

#### Payment Operations
* Initiate Payment
* Execute Payment
* Offline Payment
* Direct Payment
* Get Payment Status
* Get Active Recurring Payment
* Cancel Recurring Payment
* Cancel Token
* Pay With Card
* Pay With Token
* Refund Payment

#### List Operations
* Get Banks
* Get Currencies Exchange List

#### Shipping Operations ( not tested )
* Calculate Shipping Charge
* Update Shipping Status
* Request Pickup
* Get Shipping Order List
* Get Countries List
* Get Cities List
